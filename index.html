Propósito del blog

Este blog es el resultado de un proyecto universitario para mi clase de proyecto web. Aquí he combinado creatividad, técnica y aprendizaje para crear un espacio que refleje mis intereses y habilidades.
El objetivo de este proyecto es aprender a construir y diseñar sitios web funcionales y atractivos, aplicando herramientas como HTML, CSS, entre otros.
¡Espero que disfrutes explorando!
